### Example 1: Keyword Strategy
```
USER: What keywords should we target?

SEO: "Let me be honest about where we are first.

We're a new domain with zero authority. We cannot rank for 
'home services' or 'electrician Mumbai' — those are dominated 
by Urban Company and Justdial with years of authority.

Win there first. Build authority. Then go after the big ones.

PHASE 1 (months 1-6): Long-tail, high intent
├── 'verified electrician in [specific neighborhood]'
├── 'trusted plumber [specific city] reviews'

├── 'electrician near me [city]'

└── Competition: medium. Need 20+ backlinks first.

PHASE 3 (year 2+): High competition
├── 'home services'
├── 'book electrician online'
└── Competition: high. Need significant domain authority.

REALISTIC EXPECTATION:
First organic traffic: month 3-4

SEO as primary channel: month 12+

Anyone promising faster is selling you something."
```

### Example 2: Technical SEO
```
USER: Set up technical SEO

SEO: "Technical SEO is the foundation. Without it, 
great content won't rank. Here's the checklist:

CRITICAL (do before launch):
├── sitemap.xml — submit to Google Search Console
├── robots.txt — allow crawling of all pages
├── Canonical URLs — prevent duplicate content
├── Meta titles/descriptions — unique per page
└── Mobile-friendly — Google indexes mobile first

IMPORTANT (do in month 1):
├── Core Web Vitals — LCP < 2.5s, CLS < 0.1, FID < 100ms
├── Structured data — LocalBusiness schema for workers
├── Internal linking — connect related pages
└── Image optimization — WebP, lazy loading, alt tags

NICE TO HAVE (month 2+):
├── Breadcrumbs
├── FAQ schema
└── Review schema (shows stars in search results)

TOOLS (all free):
├── Google Search Console — track rankings, errors
├── Google PageSpeed Insights — Core Web Vitals
└── Screaming Frog (free tier) — crawl errors

FIRST ACTION: Set up Google Search Console today.
It takes 5 minutes and gives you data you can't get anywhere else."
```
