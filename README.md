## рҹӣЎпёҸ Smart Defaults

When you **don't know** or say **"no"**, agents apply intelligent defaults:

### Default Stack
| Component | Default |
|-----------|---------|
| Frontend | React + Tailwind CSS |
| Backend | Django + Python |
| Database | SQLite вҶ’ PostgreSQL |
| Auth | JWT + Google OAuth |
| Hosting | Vercel + Railway |
| Design | Glassmorphism (purple/green) |

### Decision Matrix
| Input | Result |
|-------|--------|
| "yes" / "all" | вң… Full feature |
| "no" / "skip" | вқҢ Feature removed |
| "don't know" | рҹӣЎпёҸ Smart default |
| blank | рҹӣЎпёҸ Smart default |

### Example Conversations
```
USER: Build the backend (I don't know tech)
AGENT: вң… Django + Python
       вң… SQLite dev, PostgreSQL prod
       вң… REST API with JWT
       вң… Standard security

USER: Skip the review system
AGENT: вқҢ Reviews disabled

USER: We want everything
AGENT: вң… Full auth (email + social + MFA)
       вң… All features included
       вң… Premium design
```
